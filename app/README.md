# Eat&Run Flutter App

A beautiful, modern food ordering app built with Flutter!

## Features

- 🍔 Beautiful UI matching the Eat&Run web design
- 📱 Mobile responsive
- 🎨 Smooth animations using animate_do
- 📦 Category filters
- 🛒 Shopping cart (coming soon)

## Getting Started

1. Install Flutter: https://flutter.dev/docs/get-started/install
2. Run `flutter pub get`
3. Run `flutter run`

## Design

- Uses the Eat&Run color scheme: #00b09b → #96c93d gradient
- Clean, modern card design
- Smooth animations and transitions
