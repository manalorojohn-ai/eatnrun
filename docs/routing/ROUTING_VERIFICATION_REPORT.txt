ROUTING SYSTEM ARCHITECTURE
================================================================================

REQUEST FLOW
─────────────────────────────────────────────────────────────────────────────

User Request
    ↓
.htaccess URL Rewriting
    ├─ Remove .php extension
    ├─ Rewrite to index.php
    └─ Preserve query strings
    ↓
index.php (Root Router)
    ├─ Parse request path
    ├─ Check for admin prefix
    └─ Route to appropriate handler
    ↓
User Route Handler          OR          Admin Route Handler
    ├─ Search user folders              ├─ Search admin folders
    ├─ Load matching file               ├─ Load matching file
    └─ Display page                     └─ Display admin page
    ↓
Response to User
